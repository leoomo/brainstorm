# AI 讨论助手 - 界面优化设计

**版本**: 1.0
**日期**: 2026-03-03
**状态**: 已批准 → **已实现**

---

> **注意**: 扩展已从 popup 改为 sidebar 面板形式，提供了更大的显示空间和更好的用户体验。

## 1. 设计目标

提升 Chrome 扩展弹窗的视觉品质，使其更现代、专业、有品牌感。

## 2. 配色方案

| 元素 | 颜色 |
|------|------|
| 背景 | `#F8FAFC` (slate-50) |
| 卡片/弹窗 | `#FFFFFF` |
| 主强调色 | `#0EA5E9` (sky-500) - 专业蓝 |
| 主强调悬停 | `#0284C7` (sky-600) |
| 文字主色 | `#0F172A` (slate-900) |
| 文字次色 | `#64748B` (slate-500) |
| 边框 | `#E2E8F0` (slate-200) |

## 3. 改进点

### 3.1 头部
- 更精致的品牌感
- 微妙背景色区分

### 3.2 模式选择卡片
- 添加图标区分模式
- 选中状态：边框变色 + 背景渐变微光
- 悬停：轻微上浮 + 阴影

### 3.3 模型列表
- 更好的选中视觉反馈
- 供应商标签样式优化

### 3.4 按钮
- 悬停：背景加深 + 微上浮
- 按压：`scale(0.98)` 反馈
- 禁用状态更明显

### 3.5 讨论界面
- 消息卡片更有层次
- 主持人消息特殊样式（金色/蓝色边框区分）
- 加载动画更精致

### 3.6 视图切换
- 平滑过渡动画
- 页面切换不突兀

## 4. 动效规格

- 过渡时长：`0.2s`
- 缓动函数：`cubic-bezier(0.4, 0, 0.2, 1)`
- 按钮悬停：`translateY(-1px)` + 阴影增强
- 卡片悬停：边框颜色变化

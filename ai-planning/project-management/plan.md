# AI 讨论助手 - 项目管理功能实施计划

## 1. 目标

为 AI 讨论助手添加项目制管理功能，解决历史讨论复用和多主题管理问题：
- 引入项目概念，支持创建、切换、删除项目
- 项目内可创建多次讨论（迭代）
- 首次使用时自动创建默认项目

## 2. 已确认的决策

| 决策项 | 选择 |
|--------|------|
| 数据迁移策略 | 直接使用新结构，清零测试数据 |
| 默认项目创建时机 | 首次使用时自动创建"默认项目-YYYY-MM-DD" |
| 项目内讨论标题规则 | 自动生成 "项目名 - 第N次讨论" |

## 4. 风险评估

| 风险 | 描述 | 应对措施 |
|------|------|---------|
| UI 复杂度 | 项目切换增加交互步骤 | 保持最小改动，渐进增强 |
| 存储限制 | Chrome storage 有大小限制 | 限制单个项目最大讨论数 |
